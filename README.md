# How to contribute

## Setting up

- Install LaTeX
- Sign up on [GitHub](https://github.com)
- Install [Python 3](https://www.python.org/downloads/)
- Install [Pandoc](https://pandoc.org/installing.html)
- Install [Atom](https://atom.io/) with the following extensions:
    - atom-build
    - build-make
    - pdf-view-plus
    - language-pfm
