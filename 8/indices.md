# Indices

::: Question
Simplify the following indices:

#) `mult('a^4', 'a^7', a=3)`
#) `mult('a^2', 'a^6', a=10)`
#) `mult('a^4', 'a^-2', a=6)`
#) `mult('a^-3', 'a^-9', a=5)`
#) `mult('a', 'a^3', a=9)`

::: Question
Extension - Evaluate the following:

#) `mult('a^4', 'a^-4', a=10)`
#) `mult('a^0', a=7)`
#) `mult('x^0')`
:::
